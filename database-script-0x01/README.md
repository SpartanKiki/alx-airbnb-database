# Task 2 — Database Schema (DDL)

This folder contains the SQL script that creates all database tables for the Airbnb project.

## Files

### ✔ `schema.sql`
Defines the complete relational database schema, including:
- Users
- Properties
- Bookings
- Payments
- Reviews
- Messages

The script includes:
- Primary Keys  
- Foreign Keys  
- ENUM fields  
- Constraints  
- Indexes for performance  

This schema follows the database specification and supports all main app functionality.
